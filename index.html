abstract

Finding a method to discover the geophysical properties of Martian rocks with certainty has been an issue for NASA.
With Geo-Ohm this method is achieveble,since it uses proved Archie equations.
Geo-Ohm is a developed active sensor that defines the geophysical properties of Martian rocks through electrical properties

EQUATION:

This project provides the porosity percentage of a given formation when provided with a resistance of the formation (given by the user):

Assumed Values

  • rw: The resistivity of water.
  • m: Cementation exponent.
  • n: saturation exponent.
